Telegram 最佳实践

完整的从安装 OpenCLAW 到配置 Telegram 的详细教程

1 环境准备
2 安装OpenCLAW
3 初始配置
4 创建Telegram Bot
5 配置Token
6 配对与测试

步骤 1: 环境准备

系统要求

  • Node.js ≥ 22.x
  • 内存 ≥ 2GB (推荐 4GB+)
  • 磁盘空间 ≥ 1GB (推荐 5GB+)
  • 能访问国际互联网

安装 Node.js 22

# 使用 nvm 安装 Node.js 22
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
source ~/.zshrc  # 或 source ~/.bashrc
nvm install 22
nvm use 22
nvm alias default 22

# 验证安装
node --version  # 应显示 v22.x.x

提示：国内用户建议提前配置好代理，确保能顺利访问 npm 和 OpenCLAW 相关服务。

步骤 2: 安装 OpenCLAW

推荐

方式一：一键安装脚本

# macOS / Linux
curl -fsSL https://openclaw.ai/install.sh | bash

方式二：NPM 全局安装

# 使用 npm
npm install -g openclaw@latest

# 或使用 pnpm (推荐)
pnpm add -g openclaw@latest

验证安装

# 检查版本
openclaw --version

# 查看帮助
openclaw --help

注意：如果提示 command not found，请关闭终端重新打开，或检查 PATH 配置。

步骤 3: 初始配置

启动配置向导

# 启动交互式配置向导并安装后台服务
openclaw onboard --install-daemon

配置向导步骤

1
Gateway 模式

选择 Local（99% 用户的选择）

2
认证方式

选择 Anthropic (API key) 或 OpenAI (API key)

3
默认 AI 模型

推荐选择 claude-3-sonnet（平衡速度、智能和成本）

4
配置聊天频道

使用空格键选择 Telegram（暂时跳过详细配置）

5
工作区设置

保持默认 ~/.openclaw/workspace

6
安装后台服务

输入 y 确认安装

提示：配置完成后，所有设置保存在 ~/.openclaw/openclaw.json 文件中。

步骤 4: 创建 Telegram Bot

1

找到 BotFather

在 Telegram 中搜索 @BotFather 并开始对话

搜索 "BotFather"，选择带有蓝色认证勾的账号
2

创建新 Bot

发送命令 /newbot

/newbot
3

设置 Bot 名称

按照提示输入 Bot 的显示名称（如：MyClawBot）

4

设置用户名

设置唯一的用户名，必须以 bot 结尾（如：my_claw_bot）

可用：myclaw_bot, openclaw_helper_bot
5

获取 API Token

BotFather 会回复包含 HTTP API Token 的消息

123456789:ABCdefGHIjklMNOpqrSTUvwxyz

重要：请妥善保管这个 Token，不要分享给他人！

步骤 5: 配置 Token 到 OpenCLAW

配置 Telegram Token

# 打开 Telegram 频道配置
openclaw configure --section channels.telegram

配置步骤

  1. 运行上述命令后，会打开交互式编辑器
  2. 找到 token 字段
  3. 粘贴从 BotFather 获取的 API Token
  4. 保存并退出（通常按 Ctrl+X，然后确认）

配置示例

{
  "channels": {
    "telegram": {
      "enabled": true,
      "token": "123456789:ABCdefGHIjklMNOpqrSTUvwxyz",
      "dmPolicy": "pairing"
    }
  }
}

重启 Gateway 使配置生效

openclaw gateway restart

步骤 6: 配对与测试

配对流程

发送消息给 Bot
收到配对码
批准配对
开始对话

1. 在 Telegram 中发送消息

在 Telegram 中找到你的 Bot，发送任意消息（如 "hello"）

2. 查看待处理的配对请求

# 查看 Telegram 的待处理配对请求
openclaw pairing list --channel telegram

预期输出：

Pending Pairings for telegram:
CODE         USER ID      NAME          DATE
claw-xyz     123456789    John Doe      2024-01-15...

3. 批准配对

# 使用上一步看到的 CODE 批准配对
openclaw pairing approve telegram claw-xyz

4. 验证连接状态

# 检查频道状态
openclaw channels status --probe

恭喜！配置完成

现在你可以在 Telegram 中与你的 OpenCLAW Bot 对话了。尝试发送：

  • "帮我查看当前目录"
  • "运行 ls -la 命令"
  • "总结这个文件的内容"（发送文件后）
